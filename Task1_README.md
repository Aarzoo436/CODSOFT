## 📖 Project Description

This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users through text-based conversation and responds according to predefined commands.

The chatbot can:

* Respond to greetings
* Tell jokes
* Provide study tips
* Show the current time
* Perform basic calculations
* Display chat history

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Random Module
* Datetime Module

---

## ⚙️ Features Implemented

1. Greeting responses
2. Joke feature
3. "Hasi nhi aayi" joke continuation feature
4. Study tips
5. Current time display
6. Basic calculator
7. Help menu
8. Chat history
9. Exit option

---

## ▶️ How the Chatbot Works

1. User enters a command.
2. The chatbot checks the input.
3. It matches the input with predefined commands.
4. The chatbot returns the appropriate response.
5. The conversation continues until the user enters **bye**.

---

## 💬 Sample Commands

* hi
* joke
* hasi nhi aayi
* time
* study tip
* calculate 78+43
* cricket
* calculate 87-98
* help
* bye

## ▶️ How to Run the Project
