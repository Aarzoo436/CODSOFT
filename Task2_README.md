## 📖 Project Description

This project is an AI-powered Tic Tac Toe game developed using Python. The game allows a human player to compete against an AI player.

The AI uses the Minimax algorithm to make optimal decisions and select the best possible move during gameplay.

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Math Module

---

## 🤖 AI Algorithm Used

* Minimax Algorithm

The Minimax algorithm evaluates possible game states and chooses the best move for the AI player.

---

## ⚙️ Features Implemented

1. Human vs AI gameplay
2. AI move prediction using Minimax
3. Win detection
4. Draw detection
5. Invalid move handling
6. Interactive command-line gameplay

---

## ▶️ How the Game Works

1. The human player plays as **X**.
2. The AI plays as **O**.
3. The player enters a position from 1 to 9.
4. The AI calculates the best possible move.
5. The game continues until a player wins or the match ends in a draw.
