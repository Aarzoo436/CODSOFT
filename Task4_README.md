# Recommendation System

## CodSoft Artificial Intelligence Internship

### Project Overview

This project is a Movie Recommendation System that suggests similar movies based on their content and plot descriptions.

The system uses TF-IDF Vectorization and Cosine Similarity to find movies with similar overviews and generate recommendations.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
- Jupyter Notebook

---

## Dataset

Dataset Used:

- tmdb_5000_movies.csv

Total Movies: 4803

---

## Project Workflow

1. Load Dataset
2. Select Movie Title and Overview
3. Handle Missing Values
4. Convert Text Data into Numerical Form using TF-IDF
5. Calculate Similarity Scores using Cosine Similarity
6. Build Recommendation Function
7. Generate Movie Recommendations

---

## Features

- Content-Based Movie Recommendation
- TF-IDF Text Vectorization
- Cosine Similarity Matching
- Simple and Fast Recommendation Engine
